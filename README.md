# 🍽️ API Restaurant

Uma API completa para gerenciamento de restaurantes desenvolvida durante o curso FullStack da Rocketseat.

O projeto foi construído utilizando Node.js, TypeScript, Express e Zod, com persistência de dados em PostgreSQL.

---

# 🚀 Funcionalidades

## 📦 Produtos

* Criar produtos
* Listar produtos
* Atualizar produtos
* Remover produtos

## 🪑 Mesas

* Criar mesas
* Listar mesas
* Remover mesas

## 🧾 Sessões de Mesas

* Abrir sessão de uma mesa
* Encerrar sessão
* Controlar sessões ativas

## 🛒 Ordens

* Criar ordens
* Listar ordens
* Atualizar ordens
* Remover ordens
